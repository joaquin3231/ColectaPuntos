Mecacinas:

Personaje azul:

Al precionar espacio con este personaje se activa el modo salto el cual te permitira saltar dos casillas
(en lugar de una) segun la tecla que preciones.

muestra ejemplo:
en este caso se salto hacia la derecha, pero se puede saltar por encima de cualquier pared o en cualquier direccion
OJO AL USAR EL SALTO ESTE SE DESACTIVARA

Personaje amarillo

A diferencia del anterior personaje este no puede saltar. pero cuando se preciona la tecla espacio este personaje
es capaz de romper una pared segun en la direccion que apuntes

Muestra Ejemplo;
en este caso se rompio la pared de la derecha, pero se puede romper cualquier pared en cualquier direccion
OJO AL ACTIVAR EL MINADO SE DESACTIVA EL MOVIMIENTO.
UNA VEZ MINADA UNA PARED EL MINADO SE DESACTIVARA Y SE REACTIVARA EL MOVIMIENTO.
CADA DOS PAREDES MINADAS SE DESCUENTA UN PUNTO

LOS PERSONAJES CON UN NOP:

por el momento estos estan desactivados asi que hay que esperar
